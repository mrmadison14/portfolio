How this gets built

I build with AI agents the way I build anything for production: changes pass an eval suite before they merge, and writes pass a verification checklist before they happen. The tools are Claude-based agent workflows. The guardrails are hard rules, each with a failure story behind it. The field notes below show how that works in practice.
